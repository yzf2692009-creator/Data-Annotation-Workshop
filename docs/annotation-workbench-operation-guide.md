# 标注页面真实操作文档

本文档基于当前前端路由和真实接口逻辑整理，覆盖任务中心可进入的主要标注页面。所有标注页面均应通过任务中心进入，或使用 URL 直达：

```text
?page=<页面标识>&task_id=<任务ID>
```

进入页面后，前端会根据 `task_id` 调用后端任务详情接口加载任务信息、任务标签、任务进度，并调用下一条样本接口加载当前可标注样本。

## 通用操作规则

### 任务与样本加载

1. 页面顶部展示任务编号、任务名称、任务类型、任务进度。
2. 标签列表来自任务详情中的真实标签，不再使用页面内置假标签。
3. 样本数据来自后端分配的真实数据项。
4. 如果任务加载失败、任务不存在或接口返回 404，页面会提示错误并清空标签和样本数据。
5. 如果当前任务只有 1 条数据，页面不展示“跳过”按钮。

### 任务模式

页面支持两类任务模式：

| 模式 | 页面展示 | 操作含义 |
| --- | --- | --- |
| 独立分工 | 独立分工 / 个人进度 | 每个执行人处理自己分配到的数据 |
| 协同共享 | 协同共享 / 协作进度 | 多个执行人共享同一任务池 |

具体进度以任务详情和下一条样本接口返回的数据为准。

### 通用按钮

| 按钮 | 说明 |
| --- | --- |
| 返回任务中心 | 返回任务中心页面 |
| 上一条 | 加载当前样本的上一条数据 |
| 跳过 | 跳过当前样本并加载下一条；单样本任务隐藏 |
| 提交标注 | 提交当前样本标注结果，成功后自动加载下一条 |
| 未标注 | 当前没有有效标注时展示，按钮禁用或阻止提交 |

## 图像标注

### 图像标注工作台

页面标识：`img-anno`

真实能力：

- 根据 `task_id` 加载任务详情、真实标签和当前图片样本。
- 支持矩形框标注。
- 标注框保存标签 ID、标签名称、标签颜色、坐标信息。
- 支持上一条、跳过、提交。
- 支持任务进度展示。
- 提交时至少需要 1 个标注框。

基本操作：

1. 从任务中心点击“开始标注”进入图像标注工作台。
2. 在右侧选择任务标签。
3. 在图片区域绘制标注框。
4. 检查标注列表和标签颜色。
5. 点击“提交标注”。

### OCR 图像文字标注

页面标识：`ocr-anno`

当前真实实现：通用企业级标注工作台。

真实能力：

- 加载真实任务、真实标签、真实图片样本。
- 对图片样本可拖拽绘制区域。
- 可选择文字类别标签。
- 可为标注结果填写备注。
- 支持上一条、跳过、提交。

基本操作：

1. 进入页面后确认图片样本已正常预览。
2. 在右侧选择文字类别标签。
3. 在图片中文字区域拖拽绘制框选范围。
4. 在标注结果备注中填写识别文本、方向、质量或其他说明。
5. 检查文字区域和备注内容后点击提交。

当前边界：

- 当前页面支持人工框选和结构化备注，不包含自动 OCR 识别引擎。
- OCR 文本内容需要通过标注项备注或说明区域录入。

### 姿态关键点标注

页面标识：`pose-anno`

真实能力：

- 根据任务标签加载关键点定义。
- 支持关键点打点、可见性状态、骨架连线。
- 支持上一条、跳过、提交。
- 支持任务进度展示。

基本操作：

1. 进入页面后确认任务标签是否为关键点定义。
2. 在图片上依次放置关键点。
3. 根据需要调整关键点可见状态。
4. 点击提交。

页面标识：`doc-ner`

## 音频标注

音频类页面共用音频标注工作台。页面会加载真实音频文件，生成可点击或拖动的波形预览，支持播放、暂停、定位、上一条、跳过和提交。

### 音频片段标注

页面标识：`audio-segment-anno`

基本操作：

3. 如果是有效音频，拖动选择片段，选择标签。
4. 填写转写文本。
5. 提交标注。

- 医学影像页面当前不提供 DICOM 窗宽窗位等专业影像工具。
- 深度图、热成像、雷达等页面当前不提供专用数值渲染面板。
- 遥感页面当前不提供 GIS 坐标系和瓦片地图能力。
- 缺陷检测页面当前支持通用框选和结构化记录，不提供专用批量缺陷工具。

## 多模态 / AI 标注

### 知识图谱标注

页面标识：`kg-anno`

当前真实实现：通用企业级标注工作台。

真实能力：

- 加载真实样本和任务标签。
- 可添加实体关系类标注项。
- 可通过备注记录实体、关系、属性和来源文本。
- 支持上一条、跳过、提交。

基本操作：

1. 阅读样本文本或查看样本内容。
2. 选择实体或关系标签。
3. 点击“添加实体关系”创建标注项。
4. 在备注中填写实体名称、实体类型、关系类型、属性和值。
5. 检查知识图谱结构说明后提交。

### 多模态标注

页面标识：`multimodal-anno`

当前真实实现：通用企业级标注工作台。

真实能力：

- 根据样本文件类型预览图片、视频、音频或文本。
- 可添加多模态标注项。
- 可记录跨模态说明、关联关系、判断依据。
- 支持上一条、跳过、提交。

基本操作：

1. 查看当前样本的图片、视频、音频或文本预览。
2. 选择多模态任务标签。
3. 点击“添加多模态标注”创建标注项。
4. 在备注中记录不同模态之间的对应关系、证据和判断依据。
5. 检查跨模态说明后提交。

### RLHF 偏好标注

页面标识：`rlhf-anno`

当前真实实现：通用企业级标注工作台。

真实能力：

- 加载真实样本和任务标签。
- 可添加偏好结果标注项。
- 可通过备注记录偏好选择、评分理由、排序依据。
- 支持上一条、跳过、提交。

基本操作：

1. 查看当前样本内容或待比较结果。
2. 选择偏好结果标签。
3. 点击“添加偏好结果”创建标注项。

当前边界：

- 偏好选择需要通过标注项和备注进行结构化记录。

## 提交数据说明

标注提交统一调用任务标注提交接口，核心字段包括：

| 字段 | 说明 |
| --- | --- |
| `dataset_item_id` | 当前样本 ID |
| `annotations` | 当前样本标注结果数组 |
| `type` | 标注类型，例如 `ocr`、`document_ner`、`audio_segment` |
| `label_id` | 任务标签 ID |
| `label_name` | 任务标签名称 |
| `color` | 标签颜色 |
| `note` | 备注或结构化说明 |
| `file_name` | 当前样本文件名 |

图片类通用框选结果会额外包含：

| 字段 | 说明 |
| --- | --- |
| `x` / `y` | 框左上角百分比坐标 |
| `w` / `h` | 框宽高百分比 |
| `coordinate_type` | 坐标类型，当前为 `percent` |

## 测试建议

1. 为每一种页面创建至少一个真实任务，并绑定真实标签。
2. 每个任务准备 1 条数据和 2 条以上数据两种场景，确认单数据任务隐藏“跳过”。
3. 分别用独立分工和协同共享模式进入页面，检查顶部进度是否正确。
4. 对每个页面完成一次标注提交，确认数据库中产生对应 `annotations` 记录。
5. 对 404 或无权限任务进行直达访问，确认页面不展示虚假标签或虚假样本。
